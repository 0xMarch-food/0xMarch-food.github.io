记一次实际上传流程

首次提交：

  1. 先在github创建你的仓库；
  2. 先进入你要上传的代码文件根目录并初始化（仅首次需要）；
  3. 关联远程 GitHub 仓库（仓库地址在GitHub仓库页面的Code按钮中获取）；
  4. 添加所有现有文件（先add再commit）；
  5. 推送到Github仓库。
